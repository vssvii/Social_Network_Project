Архитектура: Проект создан на архитектуре Model-View-ViewModel (MVVM)




Проект состоит из:

1) Регистрация и аутентификация пользователя через номер мобильного телефона
2) Страница профиля
3) Страница новостей
4) Страница друга
5) Страница любимых постов


1) Регистрация и аутентификация пользователя через номер мобильного телефона
